# Playbook: Bathroom Remodel

*When you need this:* homeowner wants a full bathroom gut-and-rebuild — new layout, fixtures, tile, and finishes.

## Sequence

1. Permit required. If the plan relocates plumbing fixtures or moves walls, expect LADBS plan-check (or equivalent in independent cities like Santa Monica, Pasadena, Burbank, Beverly Hills — each runs its own building dept); a like-for-like fixture swap may qualify for over-the-counter issuance. Confirm with the local building dept before bidding the schedule.
2. If pre-1978, test painted surfaces being disturbed for lead; use EPA RRP practices (certified renovator, containment, cleanup verification) if positive. Test flooring mastic, old texture, or popcorn ceiling for asbestos before demo — the AQMD Rule 1403 survey requirement applies regardless of building age; pre-1990 just makes a positive result more likely.
3. Protect adjacent rooms/hallways, shut off water at the branch or main, disconnect and cap fixtures as needed.
4. Demo: remove tile, fixtures, vanity, tub/shower, and flooring down to studs/subfloor per scope.
5. Rough plumbing: relocate or add supply and drain/vent lines per the approved plan.
6. Rough electrical: add GFCI-protected circuits near water, relocate exhaust fan wiring — fan capacity/ducting must meet Title 24/CalGreen ventilation requirements for room size.
7. Rough plumbing and electrical inspection — schedule before closing any walls.
8. Insulate exterior walls if applicable (Title 24 requirement on envelope work).
9. Hang drywall — moisture-resistant board generally, cement/glass-mat backer board at wet areas.
10. Install waterproofing membrane at shower pan/surround, tub surround — a required inspection point on most permits before tile goes up.
11. Set tile: shower/tub surround, floor, accents; grout and seal.
12. Install vanity and countertop.
13. Set fixtures: toilet, sink/faucet, tub/shower valve trim, showerhead.
14. Install exhaust fan cover and verify operation.
15. Final electrical, plumbing, and building inspections.
16. Paint, install mirror/accessories, final punch list and clean.

## Commonly missed

- Not confirming exhaust fan CFM and ducting termination meet Title 24/CalGreen ventilation requirements — inspectors check this and it's an easy fail if the fan is undersized or vented into the attic instead of outside.
- Skipping lead/asbestos testing on older homes before demo starts, which can shut a job down mid-swing if surprise material turns up.
- Scheduling tile before the waterproofing/rough inspection is signed off — some inspectors require to see the membrane before it's covered.
- Assuming a "simple" fixture swap doesn't need a permit when the scope quietly grew to include a moved drain line.

## Rough SoCal cost (2025-26)

$350-$600/sf of bathroom floor area for a full gut remodel (mid-to-upper finish level, permit + labor + materials); $150-$250/sf for a fixture-swap-level refresh.
