# Playbook: EV Charger Circuit

*When you need this:* homeowner buys an EV and needs a dedicated 240V circuit run to the garage or driveway for a Level 2 charger.

## Sequence

1. Load calculation on the existing panel to confirm spare capacity for a new 240V/40-50A (typical) circuit. Flag to the client up front: if there's no spare capacity, this job becomes a panel upgrade first, not a simple circuit add — don't quote it as a small job until this is checked.
2. Permit application: electrical permit; many jurisdictions including LADBS have a streamlined/expedited residential EV permit track, often OTC same-day.
3. Run a new dedicated 240V circuit from the panel to the charger location (garage wall or driveway-adjacent post).
4. Install breaker and wiring sized per the charger manufacturer's spec (confirm amperage and wire gauge match the actual unit being installed, not a generic assumption).
5. Mount the charger unit — wall-mount bracket or freestanding pedestal depending on location.
6. Terminate connections at both panel and charger.
7. Schedule and pass electrical inspection.
8. Test charger function and confirm it draws correctly under load without nuisance tripping.

## Commonly missed

- Quoting the circuit run without first checking panel capacity — this is the #1 scope-creep trigger on EV jobs and needs to be flagged before any price is given.
- Using the wrong wire gauge/breaker size because the charger's actual amperage draw wasn't confirmed against the manufacturer spec.
- Not accounting for conduit routing distance/difficulty (detached garage, long driveway run) which can dominate the cost more than the circuit itself.
- Missing that some HOAs or independent cities have their own EV-specific permit form even when the electrical work itself is simple.

## Rough SoCal cost (2025-26)

$800–$2,500 for a straightforward circuit run with adequate panel capacity; add $4,000–$8,500+ if a panel upgrade is required first.
