# Playbook: Solar PV Installation

*When you need this:* homeowner wants to install rooftop solar to offset electric bills, often paired with an EV or heat-pump conversion that's pushing electric load up.

## Sequence

1. Site assessment and system design — evaluate roof structural capacity, orientation/shading, and panel layout. Assess roof condition first: panels typically run a 20-25 year service life, so a roof that will need replacement within a few years should be re-roofed before panels go on, not after.
2. File the building/electrical permit application (plan-check typical for system design, though many jurisdictions including LADBS now offer streamlined solar permit tracks) and the utility interconnection application in parallel — LADWP within LA city, SCE elsewhere, under the Rule 21 interconnection process. This utility approval frequently runs longer than the permit and installation combined, so file it on day one.
3. Install roof-mount racking, flashing each penetration correctly — this is the single biggest leak-risk point in the whole job if done wrong.
4. Install panels on the racking per the approved layout.
5. Wire the inverter(s) into the electrical system and tie to the main panel; if the panel lacks capacity for the interconnection, this may require a load center addition or a panel upgrade — flag this early, not at final inspection.
6. Install the production meter/monitoring equipment.
7. Schedule and pass the building/electrical inspection.
8. Await utility inspection and Permission to Operate (PTO) — the system legally cannot be energized until PTO is granted, separate from the building department's sign-off.
9. Commission the system once PTO is issued and walk the homeowner through the monitoring app/controls.

## Commonly missed

- Treating the utility interconnection (Rule 21) as a formality instead of the actual critical path — PTO can take weeks longer than the physical install.
- Skipping a roof condition assessment before mounting, leading to a costly panel removal/reinstall for a re-roof a few years later.
- Not checking main panel capacity for the interconnection point early enough to catch a needed panel upgrade before final design lock.
- Poor penetration flashing at racking anchors — the most common source of solar-related roof leaks.

## Rough SoCal cost (2025-26)

$3.00–$4.50/watt installed for a typical residential system (e.g., $18,000–$27,000 for a 6kW system) before incentives, plus utility interconnection fees; add cost if a panel upgrade or re-roof is needed first.
