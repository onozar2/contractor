# Playbook: Garage Door Replacement

*When you need this:* an old garage door is damaged, off-track, or inefficient (uninsulated single-panel), and needs replacement with a like-for-like or upgraded insulated door.

## Sequence

1. No permit is typically required for a like-for-like door swap on the existing opening. A permit is required if the opening size is being altered, or if new electrical is run for a new opener circuit — verify with LADBS (City of LA) or the applicable local building department.
2. Before removing anything, note the torsion spring tension is high-energy stored force — this removal step is safety-critical and should only be done by someone trained on spring release; this is not a step to rush or skip proper technique on.
3. Remove the old door panels, track, and springs.
4. Inspect the header framing and track mounting surface — on older LA homes, garage door headers were often never properly braced for seismic/lateral load; flag and address inadequate header bracing before hanging new heavier track and door hardware, since this is a known soft-story-adjacent weak point.
5. Install new track and rollers, plumb and level per manufacturer spec.
6. Install new door panels — note that California's push toward Title 24 energy compliance has made insulated door panels the common spec now even on standard replacements, not just upgrades.
7. Set torsion spring tension to match the new door's weight.
8. Install and connect the opener — if this is a new circuit (not reusing an existing outlet/circuit), it needs a dedicated electrical run per code.
9. Program the photo-eye safety sensors — required by code on all residential garage door openers.
10. Test the auto-reverse safety function (door must reverse on contact with an obstruction) before calling the job complete — this is a non-negotiable safety test, not optional.
11. Weatherstrip and seal the full perimeter of the new door.

## Commonly missed

- Rushing or improvising torsion spring removal/tensioning instead of following proper high-tension procedure — this is where serious injuries happen on this job.
- Not checking/bracing an inadequately framed header before hanging heavier insulated door panels and hardware.
- Skipping the auto-reverse safety test, which is both a code requirement and a real safety issue for kids/pets.
- Assuming no permit is needed when the customer also wants the opening widened — that's a structural change, not a swap.

## Rough SoCal cost (2025-26)

$1,800–$3,500 installed for a standard insulated single-car door with new opener; $3,500–$6,000+ for double-wide or higher-end insulated/custom doors.
