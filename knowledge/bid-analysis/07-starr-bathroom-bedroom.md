# Bid Comparable: Starr Complete Bathroom & Bedroom Remodel

**Contractor:** Guideline Builders, Inc. — Arie Aflalo — CSLB License #1021305
**Date:** March 19, 2023
**Client:** Randall and Lisa Starr, Los Angeles CA 90045
**Form:** "Complete Bathroom & Bedroom Remodeling Agreement," complies with CA B&P Code §7159 and Civil Code §3097(1)

## Project type / scope size
Primary bathroom remodel (2-sink vanity, drop-in tub, separate shower, heated towel rack, in-wall storage) PLUS master bedroom, master closet, and upstairs hallway paint/finish work. Multi-room but not whole-home — a "primary suite refresh" scope. No sf given anywhere. Flag: no $/sf computable.

## Full scope of work (as written, by area)

**Demolition:**
- Per approved blueprint; job site kept clean; portable toilet + privacy fans provided (same client-amenity language as bid #02)

**Vanity and Fixtures (bathroom):**
- Install solid stone slab for vanity top
- Install 2 vanity fixtures, 2 sinks, 2 drains
- Install drop-in tub with new tub faucet + drain
- Install heated towel rack with hardwire electric connection
- Build + install cabinet for in-wall storage (toilet room)
- **Vanity provided by customer; vanity NOT installed by contractor** (unusual explicit carve-out — contractor installs the stone top/fixtures but not the vanity cabinet itself)

**Lights (bathroom):**
- Install lighted wall mirror with electrical outlet
- Replace overhead ceiling lighting

**Paint (bathroom):**
- Paint walls and ceiling
- Paint 2 doors (main door + toilet-room door) — lacquer paint (a premium paint finish call-out, notable upsell-level detail)

**Shower:**
- Install new tile in shower
- Re-plumb shower for wand-style shower head + separate mix valve
- Move shower drain to edge, install linear drain ("like girls' bathroom" — informal client-reference language kept verbatim in the contract, flag as non-standard/informal drafting)
- Install marble floor with marble baseboards
- Replace all door casings

**Master Bedroom:**
- Paint walls/ceiling, main door + door casing (lacquer), crown molding (lacquer)
- Replace + paint floor boards to match existing style
- Add 6 ceiling lights (4")
- Install new carpet

**Master Closet:**
- Paint walls/ceiling + 1 door (lacquer)
- Install new door casing to match existing style
- Move existing ceiling light + add one additional ceiling light

**Upstairs Hallway:**
- Paint ceiling, crown molding (lacquer), 2 doors + door casings (lacquer)

## Inclusions / exclusions — coverage split
**Client-supplied (explicit, longest list in the set):** tile, countertop slab/stone, appliances, sinks, faucets, interior/exterior doors, windows, **vanity**, **cabinet**, hardware, molding, floor tile, light fixture coverings, outdoor lighting (excl. pendant/chandelier), glass doors, plumbing fixtures (faucets), chimney/fireplace finishes, other fixtures unless indicated otherwise. Additionally: "**Customer will work directly with carpet company / vanity cabinet** / permits fees or any additional fees are not included in total price" — explicit carve-out that carpet and vanity-cabinet vendors are contracted directly by the client, not through Guideline Builders (same "client manages specialty vendor directly" pattern as bid #03's cabinet maker carve-out).

**Contractor-covered:** demolition, stone vanity-TOP fabrication/install (not the vanity cabinet), plumbing/electrical labor, tile installation, marble floor/baseboard installation, paint labor (including lacquer-finish doors/trim), crown molding install, ceiling light installation.

**Permits/fees — client pays:** identical boilerplate to bids #02/#03/#05/#06 — "responsible for all building permit fees, or any other city fees, inspection fees, engineering fees, and other public fees ... HERS raters & structural observation fees."

## What this teaches
Confirms, for a 5th time in this set, that Guideline Builders' standard exclusions paragraph (client supplies all finish materials; contractor supplies labor + rough construction + fixture installation) is their fixed house policy across bathroom, bedroom, and ADU contract types — strong signal this split is the de facto CA remodeling-market norm, not one contractor's quirk. The clean 5-stage payment schedule that sums to exactly 100% (contrast with bid #01's 110%-summing schedule) is the best small-remodel payment-milestone template in the set for the Joon template.
