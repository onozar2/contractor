# Bid Comparable: Neal Bathroom Remodeling Proposal — STRUCTURAL EXEMPLAR

**Contractor:** Expo Home Design & Remodeling Inc. — Jacob Hemo — CA License #911716 (12725 Ventura Blvd Suite D, Studio City, CA 91604)
**Date:** September 21, 2020
**Client:** Vernell Neal, Los Angeles CA 90008

*This is the strongest structural example in the set: clean trade-by-trade scope, a "General Provisions" section, and clear signature blocks. Used as the primary structural template for the Joon bid template (see JOON-BID-TEMPLATE.md).*

## Project type / scope size
Full bathroom remodel including new wall framing (i.e., this is a bathroom reconfiguration/expansion, not just a finish-out of an existing footprint). No sf given — flag for $/sf; a typical small-to-mid LA bathroom runs ~40-60 sf, so **assuming ~50 sf** (stated nowhere in the source, purely an external assumption for benchmarking) this prices out to roughly **$530/sf all-in labor+material** — treat this figure as a rough planning cross-check only, not a sourced fact.

## Full scope of work (as written, by trade)

**Framing:**
- Frame in new bathroom walls
- Frame in door opening; install bathroom entry door
- Frame in shower pan
- Build a sitting bench inside the shower area
- Create an opening for a shampoo niche inside the shower area

**Plumbing:**
- New shower plumbing (drain, hot/cold lines)
- New vanity plumbing (drain, hot/cold lines)
- New toilet plumbing (drain, cold line)
- Install shower head, shower valve, toilet

**Electrical:**
- Provide + install bathroom fan
- Provide + install 4 LED recessed lights
- Provide + install light switches as necessary
- Provide + install GFCI outlet

**Hot Mop:**
- Apply hot-mop material inside shower (leak prevention)

**Floating Cement:**
- Provide + install tar paper and wire mesh on bathtub/shower walls
- Provide + apply cement on shower walls and floor
- Re-float walls and floor before tile install

**Tile:**
- New tile on shower walls, floor to ceiling
- New tile on shower floor
- New tile on bathroom floor

**Drywall and Paint:**
- Provide + install wall insulation
- Provide + install drywall as needed; patch and sand
- One coat primer, two coats paint on walls and ceiling ("customer will select color and design")
- Install base molding

**Clean up:**
- Clean up and haul away debris upon completion

## Total price
**$26,580.00** — single lump sum, no per-trade breakout (unusual for the level of trade detail given — the scope reads like an itemized takeoff internally, but only a bottom-line price is disclosed to the client).

## Price per sf / fixture
Not computable from the document itself (no sf stated). See assumption-based cross-check above (~$530/sf @ assumed 50 sf) — flag clearly as an external assumption, not sourced.

## Payment schedule
**None specified.** No deposit, milestone, or final-payment structure appears anywhere in the document — this is the one clear structural weakness relative to the Guideline Builders bids, and a gap the Joon template should NOT copy (CA law requires a compliant down-payment/progress structure; see JOON-BID-TEMPLATE.md payment section).

## Inclusions / exclusions — coverage split
This bid is unusually **inclusive relative to the other 6** — it does not carry the long "client supplies all finish materials" exclusion paragraph seen in every Guideline Builders bid. The scope explicitly says "Provide and install" for insulation, drywall, fan, recessed lights — i.e., contractor-supplied — and tile/plumbing fixtures (shower head, valve, toilet) are listed under contractor scope without a material exclusion. Only paint *color/design selection* is explicitly left to the customer ("Customer will select color and design" — a selection right, not a supply obligation).

**\* General Provisions (verbatim, 4 numbered items):**
1. Area affected by the work will be vacated and emptied by the customer.
2. Work may interfere with parking spaces.
3. City fees, inspection fees, permit fees and other public body fees will be paid by the customer, if required.
4. All plumbing to be connected to existing.

**Permits/fees — client pays** (item 3 above), consistent with every other bid in this set.

**Signature block:** clean two-line format — "CUSTOMER SIGNATURE: ___ DATE: ___" / "CONTRACTOR: ___ DATE: ___" — simple and clear, worth keeping as-is in the Joon template.

## What this teaches
This is the only bid in the set that reads as a genuinely **all-in labor+material turnkey price** (apart from permits) rather than a labor-heavy price with client sourcing everything — useful upper-bound comparable for what a fully-inclusive small bathroom remodel costs ($26,580 for a bath with new framing, full waterproofing system, floor-to-ceiling tile, and finish paint). Its trade-by-trade organization (Framing → Plumbing → Electrical → Hot Mop → Floating Cement → Tile → Drywall & Paint → Clean Up) is the clearest, most buildable sequence of any bid reviewed and is the backbone structure adopted for the Joon bid template's scope-of-work section.
